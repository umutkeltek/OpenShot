# OpenShot

A free, open-source screenshot and screen recording tool for macOS.

**Why?** Because macOS deserves a powerful, free capture tool. Inspired by CleanShot X — built for the community.

## Features

- Area, window, fullscreen, and scrolling capture
- Full annotation editor (arrows, shapes, blur, pixelate, text, counters)
- Screen recording (MP4 + GIF) with audio
- On-device OCR — capture any text on screen
- Floating screenshots — pin any capture above all windows
- Quick Access Overlay — copy, save, annotate, drag & drop instantly
- Capture history with search and filters
- Fully customizable keyboard shortcuts
- Privacy-first — everything happens on your Mac, no cloud required

## Requirements

- macOS 14.0+ (Sonoma)
- Xcode 15+ (to build from source)

## Building

1. Clone the repo
2. Open `OpenShot.xcodeproj` in Xcode
3. Build and run (Cmd+R)
4. Grant screen recording permission when prompted
